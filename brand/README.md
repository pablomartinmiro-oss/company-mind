# Company Mind Brand Kit

The single source of truth for visual and verbal brand.

## Contents
- **tokens/** — TypeScript design tokens (colors, type, spacing)
- **logo/** — SVG logo variants
- **voice/** — writing guidelines for UI copy and AI-generated content
- **assets/** — downloadable PDFs, Figma exports, favicons
